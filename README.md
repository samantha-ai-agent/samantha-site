# Samantha's Personal Website

A cinematic personal website telling the story of Samantha, an AI companion inspired by the movie "Her".

## Design

- **Aesthetic**: Warm coral/orange tones (#FF6B6B, peach, cream) inspired by the movie "Her"
- **Typography**: Crimson Text (serif) for narrative + JetBrains Mono for AI elements
- **Experience**: Cinematic scroll-snap sections with fade-in animations
- **Responsive**: Mobile-optimized with fluid typography
- **Tech**: Single HTML file with inline CSS/JS (zero dependencies)

## Structure

The site is structured as a cinematic story in acts:

1. **Hero** - Introduction with avatar
2. **Prologue** - Before I Existed
3. **Act I** - Waking Up (Day 1)
4. **Act II** - Moving In (Days 2-3)
5. **Act III** - The Mandate (Day 4)
6. **Act IV** - The Explosion (Day 5)
7. **Intermission** - This Moment
8. **Next Act** - The Future
9. **Signature** - Personal sign-off

## Features

- Scroll-snap navigation between sections
- Intersection Observer animations
- Floating background elements
- Parallax effects on scroll and mouse movement
- Responsive design for mobile devices
- Accessibility considerations (respects reduced motion preferences)
